Welcome to the Menu Finder

The menu Finder is a simple interface that enables users to:

  • Find menus at specific locations,
  • Add new dishes to the database,
  • Create menus, selecting among the existing dishes, for specific locations,
  • Create new locations.

It is built using Node.js, Express and MongoDB.

You can create a new profile and register as a new user, or login using an existing profile, via the buttons below.
