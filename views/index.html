Welcome to the Menu Finder

The menu Finder is intended to be the foundation for a larger project. The goal is to create an interface that enables users to:

  • Find menus at specific locations,
  • Add dishes to the database,
  • Add dishes to the menus for specific locations,
  • Create new locations

As the program grows, I aim to cretae an app where end users can search locations for the menus of the day, and where owners can edit the menus. Dish descriptions and ingredients will also be available.
